# 上市公司深度调研方法论

基于产业-资本双重视角的系统性研究框架。

## 快速开始

执行深度调研的标准流程：

```
1. 理解方法论本质（phase-1-overview.md）
2. 执行四轴分析（phase-2-four-axis.md）
3. 识别关键变量（phase-3-research-upgrade.md）
4. 分析市场情绪（phase-4-emotion-trading.md）
5. 数据验证（phase-5-data-system.md）
6. 输出研究报告（phase-6-output-standard.md）
```

---

## 核心原则

**深度调研的本质**：不是"看更多数据"，而是构建一套**可验证、可迭代、可交易**的认知体系。

**思维范式转变**：
```
"公司赚钱吗？"    →   "公司在产业链的什么节点？"
"PE贵不贵？"      →   "什么变量会改变它的命运？"
"毛利率多少？"    →   "护城河结构是什么？"
```

## 四轴分析框架速览

### 第一轴：工程/业务维度
- 产业链拆解（工程颗粒度）
- 商业模式定位
- 工艺深度判断

### 第二轴：国家/制度维度
- 股权与治理结构
- "国家工程平台"特征识别

### 第三轴：周期维度
- 周期性识别
- 周期定位矩阵

### 第四轴：替代/成长维度
- 护城河分析框架
- 国产替代叙事

---

## 适用与不适用

### ✅ 特别适用
- 周期+成长双重属性的行业（半导体、新能源、基建等）
- 国家队/平台型公司（央企、战略新兴产业核心企业）
- 存在明显预期差的标的（表面亏损但内在改善）

### ❌ 不适用
- 纯消费股（缺乏"国家平台"维度）
- 互联网轻资产公司（缺乏"工程能力"维度）
- 完全市场化竞争行业（缺乏"制度接口"维度）
- 没有明显周期属性的行业

### ⚠️ 谨慎使用
- 金融股（周期属性过强，其他维度弱）
- 纯资源股（缺乏"系统集成"维度）
- 早期创业公司（缺乏财务数据支撑）

## 使用工作流

### 新用户快速上手

1. 阅读 [phase-1-overview.md](references/phase-1-overview.md) 理解方法论本质
2. 阅读 [phase-7-principles.md](references/phase-7-principles.md) 确认适用范围
3. 针对目标公司执行 [phase-2-four-axis.md](references/phase-2-four-axis.md)
4. 使用 [phase-6-output-standard.md](references/phase-6-output-standard.md) 输出报告

### 深度研究流程

1. **阶段一**：理解方法论定位（5分钟）
2. **阶段二**：执行四轴分析（核心，30-60分钟）
3. **阶段三**：识别关键变量（15分钟）
4. **阶段四**：分析市场情绪（15分钟）
5. **阶段五**：数据验证（20分钟）
6. **阶段六**：输出标准化报告（20分钟）

### 迭代更新

- **季度更新**：财务数据、机构持仓、股东户数
- **半年更新**：行业竞争格局、毛利率趋势
- **年度更新**：战略定位、国企改革进度、新业务拐点
- **事件驱动**：重大订单、政策变化、技术突破
