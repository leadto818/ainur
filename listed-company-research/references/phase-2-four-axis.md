# 阶段二：四轴分析框架

四轴分析是深度调研的核心方法论，从四个维度解构公司价值。

---

## 2.1 第一轴：工程/业务维度

**核心问题**：公司的真实能力边界在哪里？

### 2.1.1 产业链拆解（工程颗粒度）

将公司业务拆分为可研究的子模块：

```
示例：洁净室工程公司
├── 空气净化（HVAC）
├── 工艺管道（气体/化学品/超纯水）
├── 电力与控制系统
├── 防震/防静电系统
└── 智能运维系统
```

**关键提问**：
1. 哪些环节技术壁垒最高？
2. 哪些环节毛利率最高？
3. 公司在各环节的自主能力如何？

### 2.1.2 商业模式定位

| 类型 | 特征 | 估值逻辑 |
|------|------|----------|
| 设计主导型 | 方案制定者、技术标准输出 | 溢价型估值 |
| 施工主导型 | 重资产、人力密集 | 成本加成估值 |
| **系统集成主导型** | **跨专业整合能力** | **平台型估值** |

### 2.1.3 工艺深度判断

```
分水岭不在于规模，而在于制程深度：

≥65nm：工程能力高度商品化
28nm/存储器：工程能力开始"国家化"
先进制程/关键存储：工程能力 = 战略资源
```

## 2.2 第二轴：国家/制度维度

**核心问题**：公司的制度性位置是什么？

### 2.2.1 股权与治理结构

```
实际控制人分析
├── 央企（国家级平台）
│   ├── 优势：拿单能力、信用背书、逆周期韧性
│   └── 劣势：效率摩擦、非商业目标
├── 地方国企
│   └── 区域性资源优势
├── 民企
│   ├── 优势：效率、激励
│   └── 劣势：信用等级、资源获取
└── 外资背景
    └── 技术/客户优势，但政策敏感性高
```

### 2.2.2 "国家工程平台"特征识别

真正的国家工程平台不是"有关系"，而是：

1. **体系内身份与项目保障**
   - 关联交易占比
   - 承接国家级项目的频次与金额
   - 控股股东的产业地位

2. **承担国家战略任务**
   - 是否参与关乎国家安全的重大项目
   - 是否在关键产业链中扮演不可替代角色

3. **制度接口价值**
   - 把政策→工程化
   - 把设备→系统化
   - 把规划→可运行的物理世界

## 2.3 第三轴：周期维度

**核心问题**：公司是"周期奴隶"还是"周期中的结构赢家"？

### 2.3.1 周期性识别

```
周期来源分析：
┌─────────────────────────────────────┐
│ 下游资本开支周期（CapEx）              │
│ ├── 晶圆厂扩产节奏                    │
│ ├── 政策驱动的基建周期                │
│ └── 全球经济景气度                    │
├─────────────────────────────────────┤
│ 结构性对冲因素                        │
│ ├── 国家项目逆周期属性                │
│ ├── 信创/国产替代的刚性需求           │
│ └── 集团级工程连续性                  │
└─────────────────────────────────────┘
```

### 2.3.2 周期定位矩阵

| 类型 | 特征 | 投资策略 |
|------|------|----------|
| 纯周期β | 完全随下游波动 | 择时交易 |
| 结构α+周期β | 平台属性平滑周期 | 中长期持有 |
| **周期β+结构α+国家平台** | **多重缓冲机制** | **核心仓位** |

---

## 2.4 第四轴：替代/成长维度

**核心问题**：公司的不可替代性在哪里？

### 2.4.1 护城河分析框架

```
如果市场再多10家竞争对手，为什么要选这家公司？

护城河类型：
├── 资质壁垒（如：工程施工总承包一级）
├── 技术经验壁垒（复杂工艺环境理解）
├── 客户认证壁垒（进入头部客户供应链）
├── 资金壁垒（项目垫资能力）
└── **系统集成能力壁垒**（最关键）
```

### 2.4.2 国产替代叙事

国产替代不是"设备厂"的专利，工程平台同样受益：

```
深桑达的角色 = 
├── 国产设备的系统级验证者
├── 国产子系统的工程集成者
└── 国产方案的风险缓冲层
```

**结构跃迁路径**：
1. 成本结构改善 → 工程毛利中枢上移
2. 系统方案溢价 → 不再只是"工程费"
3. 话语权上移 → 从"执行平台"走向"方案平台"

## 2.5 四轴评估模板

```
┌─────────┬─────────┬─────────┬─────────┐
│ 工程维度 │ 国家维度 │ 周期维度 │ 替代维度 │
├─────────┼─────────┼─────────┼─────────┤
│         │         │         │         │
│         │         │         │         │
└─────────┴─────────┴─────────┴─────────┘
```

使用说明：
1. 对每一轴进行深度分析
2. 识别关键变量和核心矛盾
3. 综合四轴判断公司结构性位置
