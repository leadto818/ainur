# 附录：关键术语表

| 术语 | 定义 |
|------|------|
| CapEx | 资本性支出（Capital Expenditure） |
| EBIT | 息税前利润 |
| EBITDA | 息税折旧摊销前利润 |
| β（Beta） | 系统性风险/市场波动 |
| α（Alpha） | 超额收益/个体能力 |
| EPC | 工程总承包（设计+采购+施工） |
| TTM | 滚动12个月（Trailing Twelve Months） |
| 高低切 | 资金从高位股切换至低位补涨股 |
| 情绪锚点 | 被资金选为板块代表的龙头个股 |
| 工程纯度 | 主营业务中核心工程业务占比 |
| HVAC | 供暖、通风与空调系统（Heating, Ventilation, Air Conditioning） |
| CapEx周期 | 资本开支周期，指企业投资扩张的周期性波动 |
| 信创 | 信息技术应用创新产业 |
| CEC | 中国电子信息产业集团（China Electronics Corporation） |
| ROE | 净资产收益率（Return on Equity） |
| 龙虎榜 | 交易所公布的当日涨跌幅偏离值达到规定标准的证券买卖信息 |

## 常用缩写速查

| 缩写 | 全称 | 含义 |
|------|------|------|
| PE | Price-to-Earnings Ratio | 市盈率 |
| PB | Price-to-Book Ratio | 市净率 |
| PS | Price-to-Sales Ratio | 市销率 |
| PEG | PE-to-Growth | 市盈率相对盈利增长比率 |
| EV/EBITDA | Enterprise Value/EBITDA | 企业价值倍数 |
| DCF | Discounted Cash Flow | 现金流折现 |
| WACC | Weighted Average Cost of Capital | 加权平均资本成本 |
| CAGR | Compound Annual Growth Rate | 复合年均增长率 |
| YoY | Year over Year | 同比增长 |
| QoQ | Quarter over Quarter | 环比增长 |
| MoM | Month over Month | 月度环比 |
