# 阶段六：研究输出标准

标准化的研究输出模板，确保交付质量一致性。

---

## 6.1 核心观点摘要模板

```
【公司定位】
不是"XX工程公司"，而是"在国家战略产业中承担复杂工程能力落地的国家工程平台"

【核心判断】
公司以"国家工程平台"为基座，兼具"国产替代α"属性，并受到行业"周期β"影响

【四轴评估】
┌─────────┬─────────┬─────────┬─────────┐
│ 工程维度 │ 国家维度 │ 周期维度 │ 替代维度 │
├─────────┼─────────┼─────────┼─────────┤
│ 系统集成 │ CEC体系 │ 结构性  │ 设备验证 │
│ 主导    │ 核心平台 │ 缓冲    │ 集成者  │
└─────────┴─────────┴─────────┴─────────┘

【命运改变变量】
正向：国产设备突破、国企改革深化、半导体投资上行
负向：半导体投资深度下行、国家工程预算收缩
```

## 6.3 完整研究报告结构

```
1. 核心观点摘要
   - 公司定位（一句话）
   - 投资结论（买入/持有/观望）
   - 核心逻辑（3-5条）

2. 四轴分析
   2.1 工程/业务维度
   2.2 国家/制度维度
   2.3 周期维度
   2.4 替代/成长维度

3. 关键变量识别
   3.1 正向变量
   3.2 负向变量

4. 情绪交易视角
   4.1 板块定位
   4.2 资金流向分析
   4.3 交易策略建议

5. 数据验证
   5.1 财务数据
   5.2 资金数据
   5.3 关键信号

6. 风险提示

7. 附录
   - 关键术语
   - 数据来源
   - 更新日志
```
